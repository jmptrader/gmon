# gmon
Linux monitoring tool written in golang 

# Description
To build:

1. Setup golang
2. get termui source
    go get github.com/gizak/termui
3. run build file
    ./build

To run

	./gmon <options>

    options
        1. m - memory info
        2. c - cpu info

To exit

	q
